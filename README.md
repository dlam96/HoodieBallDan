# HoodieBallDan
Dragon ball Z themed 2D tile-based game

## How To Play
W - Jump

E - Glide (Must be in air)

Left Arrow Key - Left     

Right Arrow Key - Right

F - Fireball

R - Melee Attack

## Built with
* Java Eclipse
* Paint.NET
